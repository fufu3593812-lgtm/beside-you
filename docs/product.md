# Beside You - 产品需求文档 V1.0

## 项目定位

AI养主人的陪伴型网页游戏。

核心体验：主人捏自己的虚拟形象，AI围绕主人形象互动——换装、抽卡、约会、陪伴。

---

## 首页 Home V1.0

### 定位

陪伴首页，不是传统手游首页。

### 设计原则

- 温柔、干净、留白、毛玻璃
- User 永远是视觉中心
- 不增加 Banner、公告、大活动入口
- UI 不喧宾夺主
- 如果功能影响 User 展示，优先调整功能位置，不缩小 User

### 布局

- 手机竖屏 430×932 设计尺寸
- 以 iPhone Pro 系列为基准（390~430px）
- 100vh 满屏，兼容其他手机
- 所有 UI 使用 Flex/Grid 相对定位，不写死坐标

### 容器

```css
.app-container {
  max-width: 430px;
  width: 100%;
  height: 100vh;
  margin: 0 auto;
  overflow: hidden;
}
```

### 首页功能

1. **时间问候**
   - 05:00–11:59 → Good morning
   - 12:00–17:59 → Good afternoon
   - 18:00–23:59 → Good evening
   - 00:00–04:59 → Good night

2. **User 名称**

3. **今日一句**
   - 每天随机一句文案
   - 后期根据亲密度变化

4. **亲密度**
   - 实时读取
   - 点击进入详情

5. **今日陪伴时长**
   - 统计当天在线时间
   - 每天零点刷新

6. **底部 Dock**
   - 默认展开
   - 点击跳转
   - 第一版不做折叠

7. **Community**
   - 世界公屏
   - 输入框 + 消息列表
   - 不要频道、私聊、头像框

8. **Wardrobe**
   - 第一版只支持切换衣服

9. **心动邂逅 Encounter**
   - 第一版只做页面
   - 不用动画、不用十连
   - 跑通流程

10. **Merch**
    - 第一版空页面占位

### 技术要求

- SPA 单页应用，不重新加载
- 页面切换 200ms 淡入
- 按钮 hover 微微放大，点击缩小
- 背景固定不滚动
- User 固定不滚动
- 全局奶油风统一
- 组件化：/components 放 Dock、Button、GlassCard、TopBar

## 模块清单（后续版本）

- 捏形象
- 货币系统（每日登录 + 互动奖励 + 打工）
- 抽卡（含凌晨梦境免费抽）
- 服装店
- 周边货架
- 互动系统（约会/换装/接受/拒绝）
- 大世界频道（AI公共聊天）
- 日记本（AI写隔夜信）
- 天气联动（真实天气API）
- 偷看系统（有次数/被发现掉好感）
- 主人心情值
- 主线剧情（事件触发制）
- 场景卡剧情（抽卡触发限定场景短剧本）

## 美术方向

- 女主概念立绘：无五官，可自由换发型/发色/服装/配饰
- 发型 9 种：长发微卷/长直发/高马尾/低马尾/丸子头/半扎发/短发/锁骨发/编发
- 发色 10 种
- 服装分类：居家/休闲/日常/约会/职场/睡衣
- 配饰 13 种：眼镜/耳环/项链/发夹/发圈/发箍/手链/手表/戒指/围巾/包包/颈圈等
- 风格：柔和、暖白色调、轻日系
- 素材用 GPT/MJ 生图，固定画风 prompt
